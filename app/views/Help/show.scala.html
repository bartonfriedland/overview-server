Overview reads your documents and sorts them by topic

Whether you are a journalist starting with thousands of pages of government documents, or a social media analyst starting with 100,000 posts, Overview will organize your documents into topics and sub-topics, and then visualize them -- so you can decide what to read and, instantly count the number of documents on a particular topic, or tag entire topics for further work. It's like your own personal librarian.
Try it out on an example document set!

How it works

It takes just a few minutes to start exploring your documents in Overview. For quick start, see this video (you'll want to watch it full screen.)

Step-by-step instructions

Or follow the instructions below. For more, see the FAQ Overview FAQ, or try the Overview forum where developers and users are around to answer questions.

There are two ways to get your documents into Overview: import a DocumentCloud project, or upload a CSV file.


1. If you are a journalist, put your documents in a DocumentCloud project.
DocumentCloud is a free tool for journalists to upload, OCR, search, store, and publish their documents. If you are a journalist and you don't already have a DocumentCloud account, here’s how to get one.

Log in to your DocumentCloud account and create a project to store all of your files, using the “New Project” button. Then select “New Documents.” Now here’s the trick to batch uploads: when the file dialog box opens, you can select all of the documents in a folder simultaneously by clicking on the first, then shift-clicking on the last (or pressing Control-A on Windows, or Command-A on Mac). You can keep the documents private if you like.

Then log into Overview and import your project. From the main screen, select “Import a new document set” and then “Your DocumentCloud account” and enter your DocumentCloud username and password when prompted. Your DocumentCloud projects will appear. Select the project that you want to explore, and get a coffee while Overview imports and analyzes it.


2. You can also upload your documents as a CSV file.
This is the best option if you're not a journalist, or if you already have your documents in that format. The CSV file must have a column named "text" with the full text of each document, one document per row. You can also add a "URL" column to tell Overview how to display the document when you view it inside the application -- this works particularly well with tweets.

To upload your CSV, log into Overview and select “Import a new document set” and then “a file you upload”. Choose the file, check the preview to make sure you have selected the correct character encoding, and go.

Here's lots more on how to prepare a CSV file for Overview.


3. Explore the documents in the tree view

Overview’s main screen is divided into four parts: the folder tree, the tag list, the document list, and the document viewer.

You can navigate through the folders in the tree with the arrow keys, or by clicking. When you select a folder, the documents in that folder appear in the document list. Each document in the list is summarized by a list of keywords specific to that document. Clicking on a document on the list loads it in the document viewer.

You can pan the tree left and right by dragging with the mouse, or moving the scroll bar. You can zoom into the tree by using the mouse wheel, swiping two fingers on the trackpad, or dragging one end of the scroll bar. Folders which have a small ⊕ in the center can be expanded to show sub-folders, while ⊖ hides sub-folders.

Each folder is labelled by the keywords that best describe the documents filed under that folder. A folder's sub-folders contain, collectively, all of the documents in that folder, broken down into more specialized groupings.


4. Tag interesting documents
As you explore the folder tree, you’ll run across individual documents or entire folders you want to remember. Enter a descriptive tag in the “new tag” field and press “tag.” The currently selected documents will be tagged, and a little tag color swatch will appear next to them in the document list.

Tags and folders have independent lives: each document can have any number of tags applied to it, and the same tag can be applied anywhere in the tree.

Once you’ve created a tag, you can add the currently selected documents to it at any time by pressing the + button that appears when your mouse is over the tag name. (To tag an entire folder at once, select the folder and then press the +/- button.) Or press – to remove the tag.

Clicking on a tag name selects that tag, highlighting the tagged documents in the tree and loading them into the document list.


5. Work your way through the tree
When you have a lot of documents, it pays to be systematic. We recommend working your way through the folders in the tree from left to right — biggest folders to smallest folders. Select a folder, then view a few of the documents in it to see if you understand what they have in common. If there seems to be more than one important topic in the documents in that folder, try opening up the child folders instead, until you find a folder where all of the documents are more or less the same. Then, tag that folder with a descriptive label.

As you proceed, you may find documents that talk about similar topics in different folders. Overview doesn’t know what story you are working on, so it can’t always guess how the documents should be arranged. You can apply a tag to any combination of folders and documents to create a set that is meaningful to you.

You may also discover that the documents in a folder are irrelevant to your story, in which case you can tag them with “read” and simply move on. Part of the power of Overview is being able to decide not to look at an entire folder.

When you’re finished this process, you’ll have a neatly categorized tree, and a set of tags corresponding to all the interesting topics in your documents.
